# LiveQuiz Arena

A real-time classroom quiz platform built with FastAPI, WebSockets, React, TypeScript, and MongoDB.

## Tech Stack

### Backend
- Python
- FastAPI
- Pydantic
- Uvicorn
- WebSockets
- MongoDB

### Frontend
- React
- Vite
- TypeScript
- CSS
- Recharts / MUI
- Framer Motion
